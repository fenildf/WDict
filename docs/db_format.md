# 数据库格式

数据库为文本文件。

一般地, 每行为一条记录, 每个记录有许多数据项。

通常, 最开始的一项叫做第一项, 作为*唯一的*键。

例如:

```
key, value1, value2
"key", "value1", "value, 2"
```

其中, 引号为了转义`,`等字符。

## 词库数据

### 单词释义数据

- 每条记录第一项为单词
- 之后的项均为释义

### 单词级别数据

- 每条记录第一项为单词
- 第二项为级别, 0~5, 5表示最常见(最重要)
